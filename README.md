软件功能：可以通过账号登录进入自己的个人专属衣橱
可以通过拍照，相册上传自己的衣服图片，记录衣服种类
在搭配界面自由搭配自己的穿搭衣服
 解决：每天为自己发愁不知道穿什么衣服提供一个有趣的选择思路
 软件首页采用瀑布流展示自己上传过的衣服，可以像小红书一样“刷”自己的衣橱
 后续上线分享大厅，可以向大厅中分享自己的穿搭风格

软件已完成网页端开发和安卓手机端开发，网页地址：qctryfit.top
双端数据互通，可通过自己注册账号，实现多端数据同步


Software Features:

Users can log in to their personal wardrobe via account login.

Users can upload photos of their clothes from their album and record clothing categories.

Users can freely create outfit combinations in the styling interface.

Solution: Provides a fun and engaging way to solve the daily wardrobe dilemma.

The homepage uses a waterfall layout to display uploaded clothing, allowing users to "browse" their wardrobe like on Xiaohongshu (a Chinese social media platform).

A sharing hall will be added later, allowing users to share their fashion styles.

The software is currently developed for both web and Android devices. Web address: qctryfit.top. Data is synchronized across both platforms; users can register an account to achieve data synchronization across multiple devices.
